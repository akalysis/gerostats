Garden notes

Growing notes on ageing, statistics, public health and later life.

Garden Notes are not always finished essays. Some are early ideas. Some are growing explanations. Some are fragments that may later become talks, papers, tools or data stories. Each note has a status so readers know how developed it is.

Seedling: an early note or idea. Growing: a note being developed or revised. Evergreen: a more settled explanation or reference note. Archived: an older note kept for record.
