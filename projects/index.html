Projects

Tools, models and public-facing work connected to ageing, statistics and place.

This page collects projects connected to Gerostats: models, tools, maps, data visualisations and public-facing outputs. Some are finished. Some are prototypes. Some are part of larger research programmes.

In development

PACSim

PACSim is a population ageing and conditions simulation project. It aims to model how health, disability, multimorbidity and care need may change across England as the population ages. Gerostats will use this space to explain the ideas, assumptions, methods and outputs behind the work in plain language.

microsimulation, ageing, disability, health expectancy, uncertainty
Prototype

Metro Map

A public-facing visual project using a metro-map style to explain connected ideas in health, ageing, inequality and public policy. The aim is to make complex systems easier to navigate without pretending they are simple.

visualisation, public health, systems, communication See the current story
Planned

Care Burden Map

A planned visual and analytical project on the distribution of care need, disability and support across places. The aim is to make care visible as a population-level issue, not only a private family matter.

care, disability, place, inequality, ageing Open the current care map
